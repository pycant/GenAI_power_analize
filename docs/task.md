### 文献检索实用建议

- **关键词组合搜索**：将**问题领域**与**方法领域**的关键词组合，能有效缩小范围、提高精度。
    - **例如**：`AI Energy Consumption` **AND** `Benchmarking`；`Efficient Inference` **AND** `Diffusion Models`。
- **利用学术数据库**：
    - **综合检索**：Google Scholar, Semantic Scholar, arXiv。
    - **精准追踪**：关注顶级会议（NeurIPS, ICML, ICLR, CVPR, ACL等）中与“Efficient”、“Green”、“Sustainable”、“Benchmark”相关的论文。
    - **文献溯源**：精读高影响力论文，并系统查阅其“**参考文献**”和“**被引用文献**”，这是发现核心经典工作和最新进展的高效方法。

### 可立即开展的重点检索方向

建议您和团队可以立即从以下几个最直接相关的方向开始文献检索：
1.  **基础领域**：搜索 **“Green AI”**、**“Sustainable AI”**、**“Efficient Deep Learning”** 的综述性论文，快速建立整体认知。
2.  **评估方法**：搜索 **“LLM Benchmarking”**、**“Diffusion Model Evaluation”**、**“Audio Generation Metrics”**，了解各模态当前的评估实践。
3.  **核心创新**：搜索 **“Pareto Frontier Machine Learning”**、**“Multi-objective Model Evaluation”**，为您定义“多维效质比”寻找理论依据。
